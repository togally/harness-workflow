# 变更简报：chg-05 端到端 smoke 测试与对人文档示范

## 变更名

- 阻塞场景：注入带 `rm -rf` 的决策 → 断言退出非零、日志无记录、汇总不生成。
- chg-01/02 集成：legacy 非空场景断言 `resolve_archive_root` 返回 primary、`migrate_archive` 迁移后无警告。
- executing 阶段在本 change 目录产出 `实施说明.md`（固定模板）。
- **不**在本仓跑真 `ff --auto`（会污染本仓 runtime）。
