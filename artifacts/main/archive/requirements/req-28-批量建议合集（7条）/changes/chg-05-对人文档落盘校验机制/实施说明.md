# 实施说明：chg-05 对人文档落盘校验机制

## 实际做了什么
- Step 1/2：新增 `src/harness_workflow/validate_human_docs.py`，将 `stage-role.md` 契约 3 的 7 份对人文档映射硬编码为 `HUMAN_DOC_CONTRACT` / `REQ_LEVEL_DOCS` / `CHANGE_LEVEL_DOCS` / `BUGFIX_LEVEL_DOCS`，实现 `validate_human_docs(root, target)` 核心函数与 `format_report` / `run_cli` 输出层。

- 本 `实施说明.md` 为 chg-05 的对人文档落盘产出。
