# 变更简报：chg-06 下游模板刷新提示文档

## 变更名

- 在 README / docs 补 1~3 行提示，明确"升级 harness_workflow Python 包即可刷新 change / plan 模板"。

## 解决什么问题

- 对应 AC-10：下游仓库用户不清楚 harness change / plan 模板是从 Python 包动态读取，担心需要额外 CLI 开关才能拿到新模板。

## 怎么做

- 在 `README.md` 合适位置新增"升级 change / plan 模板"小节，写明 `pip install -U harness-workflow` 即可。
- 可选同步 `harness update --help` 描述（不改行为）。
- 不新增 CLI 子开关。

## 影响范围

- 文档：`README.md`（或 `docs/upgrade-templates.md`）。
- 可选代码：`src/harness_workflow/cli.py` 的 help text（只改描述字段）。

## 预期验证

- `grep` 能在 README 中看到 `pip install -U harness-workflow` 与"change / plan 模板"。
- 在下游仓库升级包后 `harness change "test"` 产出最新模板。
- 无 CLI 行为变化（`harness update` 不增子开关）。
