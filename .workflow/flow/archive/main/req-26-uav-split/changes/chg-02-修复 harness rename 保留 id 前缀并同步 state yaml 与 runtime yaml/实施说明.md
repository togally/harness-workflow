- 重写 `rename_requirement` / `rename_change`：目录改名保留 `{id}-` 前缀（req-NN / chg-NN），标题走 `slugify_preserve_unicode`，空格/全角冒号折叠，CJK 保留；同步改写 meta.yaml 的 title（id 不变）与 requirement.md / change.md 中旧标题。
