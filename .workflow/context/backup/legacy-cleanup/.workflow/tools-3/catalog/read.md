# 工具：Read

**类型：** 内置工具
**状态：** active

## 适用场景
- 读取已知路径的文件内容
- 读取状态文件（runtime.yaml、session-memory.md）
- 读取规则文件、角色文件

## 不适用场景
- 不知道文件路径时（用 Glob 先找）
- 搜索文件内容中的特定字符串（用 Grep）

## 推荐 prompt 模式

| 任务类型 | 调用方式 |
|---------|---------|
| 读取状态 | 指定完整路径，limit 参数按需设置 |
| 读取大文件 | 使用 offset + limit 分段读取 |

## 注意事项
- Edit 操作前必须先 Read（工具要求）
- 避免重复读取同一文件（浪费 context）
